# Inventory Management System (Java)

## 🚀 Features
- HashSet for unique SKU
- TreeSet for sorted products
- LinkedList for transaction history
- Stack for undo operations
- Queue for low stock alerts
- Comparable & Comparator sorting
- Generic utility methods

## 🛠️ Technologies
- Java Collections Framework
- OOP Concepts

## ▶️ How to Run
1. Open in VS Code
2. Compile:
   javac -d . src/com/inventory/Main.java
3. Run:
   java com.inventory.Main

## 📊 Data Structures Used
- HashSet → uniqueness
- TreeSet → sorted order
- LinkedList → history
- Stack → undo
- Queue → alerts
